Quake Live

Quake Live should work out well. Buy the game and you are all set. Maps download very fast from the server. Customizing the game is a lot easier than with Quake 3 Arena. No fancy commands needed to customize skins, FOV, picmip, etc. Make changes in the GUI.

Q3A is not compatible with QL. You must purchase Quake Live on Steam.

If for some reason QL doesn't work out, let's switch to Q3A on Saturday. I can share out Q3A if needed.

Launching Game Issue

Some people are having issues opening the game. A gray screen flashes and the game doesn't open. There are a few workarounds you can try. Any single workaround should work.

  • Go to the installation folder of QL. Run quakelive_steam.exe as administrator
  • Disable your AV and then open the game
  • Open the game. As soon as it goes gray and back to the desktop, open it again.
